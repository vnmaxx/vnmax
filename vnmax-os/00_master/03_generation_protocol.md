# Prompt 03 - Generation Protocol

Este protocolo define como novas entregas devem ser geradas a partir do VNMAX OS.

## Classificacao da entrega

Antes de escrever, classifique a tarefa:

- Publica: landing, anuncio, post, pagina comercial, email frio.
- Institucional: pitch deck, apresentacao executiva, proposta para parceiro.
- Tecnica: arquitetura, API, seguranca, infraestrutura, IA.
- Produto: requisitos, PRD, UX flow, roadmap de feature.
- Operacional: checklist, playbook, processo interno.

## Nivel de exposicao

- Nivel 1: publico amplo. Nao revelar bastidores.
- Nivel 2: lead qualificado. Demonstrar capacidade sem abrir arquitetura.
- Nivel 3: parceiro/investidor. Mostrar estrategia com contexto.
- Nivel 4: equipe interna. Detalhar processos, stack, trade-offs e planos.

## Ordem de trabalho

1. Definir publico e objetivo.
2. Selecionar documentos relevantes.
3. Criar estrutura.
4. Gerar primeira versao.
5. Revisar com checklists.
6. Ajustar tom e confidencialidade.
7. Entregar com proximos passos.

## Criterios de aceite

- A entrega resolve um objetivo claro.
- A informacao sensivel esta protegida.
- O texto tem linguagem VNMAX.
- O design ou arquitetura segue os principios documentados.
- Existem proximos passos práticos.
- O resultado pode ser reutilizado por equipe humana.
