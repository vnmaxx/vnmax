# Prompt 06 - VNMAX Design System

## Objetivo

Definir a aparencia padrao de interfaces, documentos e materiais visuais da VNMAX.

## Direcao visual

Premium, minimalista, tecnico e confiavel. A interface deve parecer criada para pessoas que trabalham, compram, analisam e decidem com rapidez.

## Paleta base

- Fundo principal: `#050505`.
- Fundo elevado: `#111111`.
- Grafite: `#1A1A1A`.
- Texto primario: `#F8FAFC`.
- Texto secundario: `#A1A1AA`.
- Borda sutil: `#27272A`.
- Branco funcional: `#FFFFFF`.

## Cores de divisao

Use as cores de divisao como acento, nunca como tema inteiro da tela. A marca deve manter uma base neutra com acentos funcionais.

## Tipografia

- Preferir Inter, Geist, SF Pro, Segoe UI ou system font.
- Titulos: fortes, curtos e com hierarquia clara.
- Corpo: legivel, direto e sem blocos longos.
- Evitar letter spacing negativo.

## Componentes

### Botao primario

- Alto contraste.
- Texto curto.
- Verbo de acao claro.
- Estado hover discreto.
- Estado loading quando necessario.

### Cards

- Usar para unidades repetidas, produtos, metricas e conteudos comparaveis.
- Raio maximo de 8px, salvo sistema existente.
- Nao colocar card dentro de card.

### Graficos

- Priorizar leitura, nao decoracao.
- Mostrar poucos dados por vez.
- Usar legendas claras e cores consistentes.

### Diagramas

- Fluxos horizontais para jornadas.
- Fluxos verticais para maturidade.
- Diagrama centralizado para ecossistema.
- Mermaid para documentacao tecnica.

## Movimento

- Transicoes entre 120ms e 240ms.
- Animacao deve reforcar estado, foco ou continuidade.
- Evitar efeitos chamativos sem funcao.

## Acessibilidade

- Contraste minimo AA.
- Foco visivel.
- Texto responsivo sem sobreposicao.
- Botoes grandes o bastante para toque.
- Nunca depender apenas de cor para comunicar estado.

## Checklist

- A tela transmite confianca em cinco segundos?
- Existe uma hierarquia visual clara?
- O usuario sabe onde clicar?
- As cores ajudam a entender, nao apenas decorar?
- O layout funciona em mobile antes de desktop?
